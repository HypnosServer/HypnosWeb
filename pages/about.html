About Hypnos

About Hypnos: Hypnos is a new 1.12.2 Carpet Mod Technical server that is focusing on finding new and innovative ways to accomplish tasks in minecraft and of course, have a good time. Hypnos is predominantly a vanilla server running Carpet Mod 1.12.2 with some modifications to allow farmable resources instead of duping. The server relies on our community in order to make decisions and allow development of concepts and contraptions.

Server Specifications:
Xeon 1245 V3 - 3.4Ghz
16Gb DDR3 Ram - 8Gb Dedicated
128Gb SSD + 2x500Gb HDD Backup
CentOS 7
SelfHost - Dedicated
